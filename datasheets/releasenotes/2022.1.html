多孔微结构分析 (PuMA)

用于多孔介质微结构的有效材料性质计算和材料响应模拟的综合工具集。

为了计算有效材料性质并在多孔介质微结构上执行材料响应模拟，此软件版本包括 NASA 的多孔微结构分析 (PuMA) 软件的首次集成。

PuMA 能够生成模拟现实材料的人工微结构，并计算形态特性，如孔隙率、体积分数、孔径和表面积。其他功能包括从连续体确定有效导热率、渗透率、曲率和扩散率，以及确定局部材料方向的技术。

主要功能

  • 生成随机纤维和随机球体标本
  • 计算连续体曲率、渗透率、导热率
  • 确定局部方向
  • 发布曲率浓度场、渗透率速度和压力的 3D 图像
  • 发布导热率的温度场和通量

样本生成器

您可以使用 PuMA 的样本生成器生成随机纤维和随机球体样本。您可以为数据集选择大小和有效孔隙率，以及指定其他参数，如纤维的半径、长度、方向和交叉。球体的参数包括直径和交叉。

求解器

PuMA 求解器的输入包括生成的随机纤维和球体样本，以及保存为多 ROI 的现实材料和分割的数据集。您可以计算连续体曲率、渗透率、导热率，以及确定局部方向。此外，您可以发布曲率浓度场、渗透率速度和压力的 3D 图像，以及导热率的温度场和通量。

系统要求

以下是使用 PuMA 的推荐最低要求，具体取决于所研究的材料性质：

  • 16-32 GB RAM 用于小型至中型模拟
  • 32+ GB RAM 用于更大型模拟

许可

此数值模拟和材料性质表征功能，正在持续开发中，可供非商业许可持有者使用，也包含在 Dragonfly 2022.1 版本的商业许可持有者中。但是，商业许可持有者需要升级其当前许可证以继续在未来版本的 Dragonfly 中使用 PuMA。请联系销售部门了解有关许可条款和费用的所有问题。

Noise2Void 深度学习降噪模型

革命性的深度学习降噪方法，直接在要降噪的数据上进行训练。

训练用于降噪的深度学习模型通常依赖于将高噪声输入图像与低噪声输出图像配对，或者使用一种称为 Noise2Noise (N2N) 的方法使用嘈杂图像的独立对。如果无法获取低噪声或嘈杂的训练目标，这些方法可能会受到限制，这在生物医学图像研究中经常是这种情况。作为训练用于降噪的深度学习模型的额外选择，此 Dragonfly 版本包括应用一种称为 Noise2Void (N2V) 的方法的选项，其中训练直接在要降噪的数据上进行。

Noise2Void 降噪的优势

  • 其他经典深度学习降噪模型，如噪声到低噪声和噪声到噪声，如果无法获取所需的输出，可能无法使用。对于 Noise2Void 模型，每个像素都被同一图像中的另一个像素替换。
  • 训练 N2V 模型通常不需要很多训练轮次。
  • N2V 模型非常适合减少椒盐噪声。

您应该注意，在某些情况下，N2V 可能无法超过具有更多训练信息的其他方法。例如，如果存在结构化噪声，降噪性能可能会下降。

选择 人工智能 > 自定义深度模型架构 > Noise2Void 以打开 Noise2Void 对话框。该对话框包含多个选项卡 — 应用和训练 — 让您配置和完成模型训练工作流程，以及将训练好的模型应用于降噪数据集。

训练选项卡上的选项可让您生成新模型，然后在选定的数据集上对其进行训练。您应该注意，大多数功能 — 模型、输入、数据增强、训练参数和预览 — 与深度学习工具对话框中可用的功能相同。

U-Net SR 深度学习超分辨率模型

优化的超分辨率网络架构，提供更高性能和更快推理。

"超分辨率"是指通过提高其分辨率来改善图像质量的过程。在大多数情况下，经过训练的超分辨率模型可以将图像从低分辨率 (LR) 转换为高分辨率 (HR)，同时保持清晰的边缘并保留重要细节。

此软件版本包括一个修改的 U-Net 网络架构 — U-Net SR — 用于超分辨率，采用混合梯度损失，减少参数数量。与其他超分辨率算法相比，此方法可提供更好的性能并加速推理。

U-Net SR 可在超分辨率架构的模型生成器对话框中使用。您应该注意，已为此模型预配置了一组初始训练参数，在大多数情况下应该效果良好，并且您的训练数据必须包括是输入图像的精确倍数的高分辨率输出图像，例如 2 倍、4 倍、6 倍等。设置模型参数时，您需要输入输出图像的缩放值。

注意

只有满足选定缩放值的数据集才会在训练并选择训练数据时出现在输入选项卡的输出下拉菜单中。

深度学习更新

Dragonfly 深度学习工具的多项增强和新功能。

Dragonfly 的深度学习工具更新包括可选择的类权重、培训和模型生成器对话框的更改，以及将绘制图表和视觉反馈图像序列导出为视频文件的选项。

可选择的类权重

在某些情况下，训练数据集可能存在显著的类别不平衡，或者所有相位可能具有相似的灰度值。为了帮助使用深度学习工具改进此类情况下的语义分割模型训练，此软件版本允许您选择数据比例和自定义类权重，以解决类别不平衡问题并加速训练。

类权重可在模型概述面板的类别框中选择。

类权重 描述
统一 无论标签如何，都均匀应用类权重。
从标签计算 类权重将在训练时重新计算，作为训练数据中标签的反比例。
自定义 让您设置自定义类权重。如果没有很多标签可用，您可以增加权重。您可以降低背景和/或不重要特征的权重。

训练进度信息

现在，训练对话框包括有关正在训练的模型和模型输入的详细信息，可提供视觉反馈以监控训练。信息框包括模型描述、训练参数、数据增强设置和有关训练数据的信息。

导出视觉反馈

现在，对于启用了检查点缓存和视觉反馈选项的模型，可以从训练结果对话框中导出图表和视觉反馈图像序列作为视频文件。

模型生成器

此软件版本包括对模型生成器对话框的细微重构。输入维度选项现在列为 2D、2.5D 和 3D。

分割向导

简化和增强的图像分割工作流程。

此软件版本中，分割向导有多项新功能和更新。

帧角色分配

帧的角色分配已扩展，包括"未使用"、"训练"、"验证"和"评估"选项。之前，这些角色仅限于"混合"和"监控"。

设置 描述
未使用 不会考虑该帧。
混合 该帧将被随机分为训练、验证和评估补丁。这是帧的默认设置。
训练 该帧将专门用于学习模型参数。
验证 该帧将专门用于训练期间的模型验证。仅适用于深度模型训练。
评估 该帧将用于计算模型分数。
监控 该帧将用于在训练期间显示视觉反馈。注意：只能将一个帧指定为视觉反馈。如果将第二个帧设置为"监控"，原始监控帧将自动重置为"混合"。

帧统计

数据比例类权重

标量生成器更新

用于表征和测量物体属性的新测量方法，以及用于交叉索引的功能。

此软件版本中，标量生成器提供了一些新的测量方法来表征和测量物体属性，以及用于交叉索引的功能。

粗糙度估计

Dragonfly 2022.1 提供了一些新的 3D 测量方法，用于估计准球形粒子或物体的表面"粗糙度"。粗糙度在各种过程中起着重要作用，例如粉末的质量保证，可以定义为真实表面与其理想光滑形式的偏差。

右键单击多 ROI，然后在弹出菜单中选择标量生成器。粗糙度估计器以及凸包表面积测量在基本测量类别中可用。

球度估计

作为表征粒子 3D 形状的附加指标，此 Dragonfly 版本提供了利用粒子体积与表面积之间关系的球度估计器。这个比率可以快速估计 3D 球度。

Feret 测量

现在，单切片多 ROI 的标量生成器中提供了最大和最小 Feret 以及 2D Feret 角度的测量。您应该注意，Feret 是从构成每个标记对象的生成凸包的体素中心计算的。

交叉索引

对于带有标量值的多 ROI，新的交叉索引选项允许您将所选多 ROI 类的标签号交叉索引到另一个多 ROI 的相交类。

数据集矢量场

从数据集生成矢量场，用于分析模拟结果和流动数据。

此软件版本提供了从数据集生成矢量场的选项。例如，进一步检查从 Dragonfly 新的多孔微结构分析 (PuMA) 模块中发布的渗透率模拟创建的压力和速度数据。

一个从 PuMA 渗透率模拟生成的矢量场如下所示。

创建矢量场

  1. 右键单击您要用于生成矢量场的数据属性和设置面板中的数据集，然后在弹出菜单中选择从数据集生成矢量场。注意：您需要选择至少两个数据集才能访问菜单项。
  2. 在选择矢量场组件对话框中，选择将提供 X、Y 和 Z 矢量的数据集。
  3. 在矢量模量下拉菜单中选择将提供矢量模量的数据集。
  4. 根据需要设置采样率。
  5. 单击确定以生成矢量场。生成的矢量场作为数据属性和设置面板上的新项目出现。您可以在 3D 视图中检查矢量场。

各向异性矢量场

用于确定骨骼结构特征与力学功能之间关联的 3D 建模工具。

3D 各向异性建模可以帮助确定观察到的结构特征与骨骼的力学功能之间的关联。各向异性是方向依赖性的属性，以 3D 矢量场形式呈现，其中颜色刻度表示大小或方向，密度是可配置的。下面显示了为分割的松质骨计算的矢量场示例。

右键单击数据属性和设置面板中所需的感兴趣区域或网格，然后选择 3D 建模 > 创建各向异性矢量场，打开对话框。

设置 描述
算法 表面法线算法基于构建由一组垂直于网格面的矢量填充的表面网格，其大小与局部网格面积成比例。
采样 让您选择采样设置：
区域框... 定义用于计算矢量场的区域。
间距... 确定分析区域中随机点之间的距离。
网格大小... 指示当前网格大小，计算为：区域框长度/间距。
邻居 让您选择采样设置：
影响半径... 定义内部将评估各向异性的内核大小或基本体积。
各向异性评估... 让您选择评估方法 — 基于投影或基于特征值。
网格平滑 确定在计算各向异性之前，从输入 ROI 获得的网格将平滑的次数。
骨骼分析模块

Dragonfly 还提供了专用的骨骼分析模块，包括皮质骨和松质骨的自动分割引导工作流程、形态测量、各向异性测量的其他选项，以及体积分数映射。

图形曲率分析

用于测量多孔介质中路径偏离的分析工具。

作为计算图形的新功能，您可以分析选定输入和输出框之间的图形曲率和喉部加权曲率。您应该注意，曲率是多孔介质偏离由直的平行毛细管组成的理想系统的度量。曲率分析和喉部加权分析结果 — 平均值、最小值、最大值和标准差 — 如下所示。

分析图形曲率

  1. 添加定义要分析的图形段的输入和输出的框形状。
  2. 右键单击数据属性和设置面板中所需的图形，然后在弹出菜单中选择图形曲率分析。
  3. 选择所需的输入和输出框。
  4. 单击确定以计算所选段内的曲率测量。

分析图形喉部加权曲率

  1. 添加定义要分析的图形段的输入和输出的框形状。
  2. 右键单击数据属性和设置面板中所需的图形，然后在弹出菜单中选择图形喉部加权曲率分析。
  3. 选择用于计算图形的感兴趣区域，以及所需的输入和输出框。
  4. 单击确定以计算图形所选段内的喉部加权曲率测量。

CT 重建

增强的 CT 重建功能，支持并行束投影和锥束投影。

此软件版本具有 CT 重建对话框的重构，以及 RTK 的集成，用于重建并行束投影和添加 SART 用于重建锥束投影。您应该注意，RTK 是基于 Insight Toolkit (ITK) 的开源软件包。

选择工作流程 > CT 重建，打开 CT 重建对话框。

输入图像... 让您选择要重建的投影数据集。
几何采集... 让您从文本文件导入几何采集参数或手动输入几何信息。
重建引擎... 让您选择重建引擎 — RTK 或 TomoPy — 以及用于重建投影数据的算法和其他设置。您应该注意，此软件版本包括同时代数重建技术 (SART) 迭代算法的实现，用于重建锥束投影。
此外，RTK 重建引擎和 FDK 算法现在可用于并行束重建。

查找旋转中心... 打开 CT 旋转查找器对话框，您可以在其中找到并应用最佳旋转中心偏移。
预处理... 预处理选项让您选择应用平场校正、中值滤波和其他校正以改善图像质量。可用选项现在包括多点分段平场校正选项，用于去除可能影响基于 CT 的测量的环形伪影。
预览... 让您以所选设置计算预览，以及从所选预览导入设置。
重建... 让您以所选设置重建投影。您可以选择自动将计算的重建加载到 Dragonfly 中，或者重建并保存所选数据集。如果您的系统内存不足以加载重建数据集，则可能需要后者。

Dragonfly 2022.1 中的各种其他增强和改进。

MC X-Ray

对于 MC X-Ray，此软件版本包括显示 BSE 和 EDS 图的结果的选项。您应该注意，MC X-Ray 让您可以模拟材料中的电子散射，以便在定量 X 射线微分析中将 X 射线发射与组成相关联。通过模拟 X 射线光谱，可以建立执行特定分析的最佳条件，以及建立检测限或探索可能的峰重叠。

切片注册

Dragonfly 的切片注册模块有一些更新，可让您自动对齐 3D 图像堆栈内的切片。您应该注意，构建一致的图像堆栈有助于确保后续处理和分析的准确性。

新的注册方法"金字塔方法"现在可用于注册图像切片。此注册方法通过最小化参考和目标图像切片之间的强度均方差，提供自动子像素注册。

基于特征的注册方法现在可以使用 SIFT（尺度不变特征变换）特征检测器。在这种情况下，应用关键点检测器和提取器来检测稳定的关键点并选择最强的特征。在许多情况下，SIFT 可以在杂乱和部分遮挡中识别关键点，因为它对统一缩放、方向、照明变化具有不变性，并且对仿射失真部分具有不变性。

报告生成器

作为 Dragonfly 报告生成器的更新，您现在可以创建包含和总结来自多个分析的结果的摘要报告。

选择实用工具 > 报告生成器，打开报告生成器对话框。然后，您可以为多个样本添加分析结果，选择摘要报告模板，然后生成摘要报告。

H5 文件导入器

对于 Dragonfly Pro 用户，此软件版本包括导入使用 Xnovo 的 GrainMapper3D 软件（*.h5 扩展名）创建的文件的改进。

图像加载器

Dragonfly 的图像加载器现在支持 JPEG 2000，以及导入以 ORS Object 文件格式保存的图像数据的预览和设置。

布局面板

Dragonfly 2022.1 版本的更新布局面板包括一组更全面的按钮，用于在选定场景中设置视图的数量和类型。

3D 可视化

此软件版本中有许多新选项和设置，可帮助优化 3D 可视化。

  • 虚拟地板面板：此软件版本具有新的虚拟地板面板，可让您微调地板设置。
  • 3D 预设：此软件版本为核心样本和有机材料提供了许多新的 3D 预设，以及双色预设，让您可以使用黑色背景创建戏剧性图像。
  • 剪辑框首选项：现在可以使用新的首选项"显示标题"在 3D 视图中显示或隐藏剪辑框的轴标题。

ROI 绘制工具

对于编辑自动分割的结果和其他任务，ROI 绘制面板上现在提供了新的"点击填充"笔刷。此工具根据未标记像素或体素的连接性，自动填充区域内的封闭区域。

多 ROI

此软件版本中，多 ROI 有许多新选项，包括将类直方图导出到基于行的 CSV 文件、生成所有标记对象的质心点集，以及交叉索引。

注释

此软件版本包括尺子、路径和形状的更新和新选项。

  • 标题：对于尺子、角度、区域和其他注释，您现在可以使用增加字体大小和减小字体大小按钮，为每个注释调整标题的字体大小。
  • 尺子属性：您应该注意，在此软件版本中，投影尺子的方向现在描述为"左/右"和"上/下"，可选择。
  • 将当前视图法线对齐到尺子：尺子弹出菜单中可用，"将当前视图法线对齐"选项让您自动将当前视图对齐到所选尺子的法线。
  • 从路径生成管状网格：此软件版本提供了快速有效地将路径转换为管状网格的选项。

宏

此软件版本中，Dragonfly 的宏播放器有多项更新，包括简单双击编辑步骤标题和/或描述、切换动作（例如播放、停止等）以及在宏中包含深度学习模型推理的选项。

公开可用的源代码

TensorFlow

Dragonfly 的许多人工智能应用程序，例如深度学习工具、分割向导和使用 AI 分割，都使用 TensorFlow API 来促进深度神经网络的训练和推理。TensorFlow 是由 Google Brain 团队创建的免费开源库，广泛用于深度学习应用程序和传统机器学习。

用户可以从 https://github.com/tensorflow/tensorflow/releases/tag/v2.4.1 自由下载和检查 TensorFlow 源代码。

Qt

Dragonfly 是使用 Qt 设计和开发的，Qt 是一个用于创建图形用户界面和跨平台应用程序的免费开源工具包。Qt 是由 Qt Company（一家公开上市的公司）开发的，可在商业许可证和开源 GPL 2.0、GPL 3.0 和 LGPL 3.0 许可证下使用。

用户可以从 https://download.qt.io/official_releases/qt/5.12/5.12.9/ 自由下载和检查 Qt 源代码。

多孔微结构分析 (PuMA)

Dragonfly 中集成的多孔微结构分析 (PuMA) 软件是由 NASA 的研究人员开发的，用于计算多孔介质数字化微结构的有效材料性质并执行材料响应模拟。PuMA 能够生成模拟真实材料的人工微结构，并计算简单的形态特性，如孔隙率、体积分数、孔径和比表面积。其他功能包括确定有效热导率和电导率（包括辐射和固体导电性——包括模拟后者的局部各向异性的能力）；从连续体到稀薄体系的有效扩散率和曲率；确定局部材料方向的技术，以及材料的力学性质（弹性系数）和渗透率。

PuMA 在 NASA 开源协议 (NOSA) 条款下可用，版权归美国政府所有，由国家航空航天局局长代表。

可以在官方 NASA GitHub 存储库中找到 PuMA 的稳定版本：https://github.com/nasa/puma

最新开发代码可以在开发 GitLab 存储库中找到：https://gitlab.com/jcfergus/puma-dev
